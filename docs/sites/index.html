How to read the network

Spatial coverage

Sites are distributed across formations and regions that capture ecological turnover in dry ecosystems.

Environmental gradients

Climate, soil, vegetation structure and landscape context should be interpreted jointly with plot-level data.

Long-term comparability

Repeated censuses make it possible to connect local dynamics with biome-wide patterns.
